📌 Project Overview

The Random Quote Generator is a simple Python project that displays a random inspirational, motivational, or funny quote each time the program runs. It uses Python's built-in random module to select a quote from a predefined list.

🚀 Features
Generates a random quote on each execution.
Easy to customize by adding your own quotes.
Beginner-friendly Python project.
Lightweight and fast.


🛠️ Technologies Used
Python 3
Random Module
